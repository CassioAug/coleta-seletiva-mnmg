# Coleta Seletiva

Pequeno site estático para prototipar pontos de coleta voluntário, utilizando mapa (Leaflet.js) e formulário.

> IMPORTANTE: é um protótipo para demonstração — não é um serviço oficial e não
> guarda respostas localmente.

## Editar pontos do mapa:

Abra `js/marcadores.js` e atualize o array `pontosColeta`.
